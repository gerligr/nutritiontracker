# Installation and Running Instructions

## HTTP-server setup

~~~
npm install http-server -g
~~~

## Run Application

Navigate to project root folder and then...
~~~
http-server -p 8102
~~~
...or...
~~~
./runClient.sh (Linux)
~~~
...or...
~~~
runClient.bat (Windows)
~~~
...and navigate to...
http://localhost:8102/index.html

Default credentials:
+ Username: admin
+ Password: password
